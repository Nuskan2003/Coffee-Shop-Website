# ☕ Modern Coffee Shop Website

A modern, responsive coffee shop landing page built using HTML, CSS, and JavaScript.

## Features

- Modern responsive design
- Hero section
- Featured coffee products
- Interactive order form
- Pricing table
- About section
- External coffee information links
- Smooth animations (Coming Soon)

## Technologies Used

- HTML5
- CSS3
- JavaScript
